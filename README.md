# frontend
## Set-up
### Installation
1. Clone the repository
2. Install required dependencies
```commandline
yarn install
```
3. Run the server - [instruction](https://github.com/LlamaLangApp/backend/edit/main/README.md)
4. Run the app
```commandline
yarn start
```
